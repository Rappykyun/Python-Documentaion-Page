Introduction

Introduction to Python

Python is a powerful and versatile programming language widely used for various tasks, including:

  • Web Development: Building websites and applications.
  • Data Science and Machine Learning: Analyzing data, extracting insights, and creating intelligent systems.
  • Automation: Automating repetitive tasks, improving efficiency.
  • Scripting: Creating small programs to perform specific actions.

Why Choose Python? Python offers several advantages that make it an excellent choice for beginners and experienced programmers alike:

  • Easy to Learn: Python's syntax is clear and readable, resembling natural language, making it easier to pick up compared to some other languages.
  • Extensive Standard Library: Python comes with a rich library of built-in modules and functions for various functionalities, reducing the need to write code from scratch.
  • Large and Active Community: Python boasts a vast and supportive community that provides extensive learning resources, tutorials, and forums for assistance.
  • Wide Range of Powerful Libraries: Beyond the standard library, Python offers a plethora of powerful third-party libraries specializing in specific domains like data science (NumPy, Pandas), scientific computing (SciPy), and machine learning (Scikit-learn).
Building Blocks

Variables and Data Types:

In Python, variables store information like numbers or text. You assign values to variables using the = operator.

Python offers various data types to represent different kinds of information:

  • Integers: Whole numbers (e.g., 10, -5).
  • Floats: Numbers with decimals (e.g., 3.14, -2.5).
  • Strings: Text data enclosed in quotes (e.g., "Hello, world!", 'This is a string').
  • Booleans: Represent True or False values.
                # Assign an integer (whole number) to a variable
                    age = 25
                    
                    # Assign a decimal number (float) to a variable
                    pi = 3.14159
                    
                    # Assign text data (string) to a variable
                    name = "Alice"
                    
                    # Assign a boolean value (True or False)
                    is_sunny = True

Operators

Operators perform calculations or comparisons on data.

  • Arithmetic operators: +, -, *, / for basic arithmetic.
  • Comparison operators: == (equal to), != (not equal to), < (less than), > (greater than), <= (less than or equal to), >= (greater than or equal to).
  • Logical operators: and, or, not for combining conditions.

                # Arithmetic operations
                    sum = 10 + 5  # Addition
                    difference = 15 - 3  # Subtraction
                    product = 4 * 2  # Multiplication
                    division = 10 / 2  # Division (returns a float)
                    
                    # Comparison operators
                    is_equal = 10 == 10  # Equal to
                    is_greater_than = 7 > 5  # Greater than
                    
                    # Logical operators (combine conditions)
                    is_adult = age >= 18 and is_sunny  # Using "and"
Control Flow

Conditional Statements

  • Make decisions based on conditions using if, elif, and else.
  • if checks a condition. If true, the following code executes.
  • elif allows checking multiple conditions sequentially.
  • else executes code if none of the above conditions are true.
                
                    age = 12

                    # Check if age is greater than or equal to 18 (adult)
                    if age >= 18:
                        print("You are an adult.")
                    else:
                        print("You are not an adult.")
Data Structures

Strings:

  • Hold text data. Access individual characters and combine strings with +.
                # Accessing individual characters (indexing starts from 0)
                    first_letter = name[0]  # Accesses the first character ("A")
                    
                    # Combining strings
                    greeting = "Hello, " + name + "!"
                    print(greeting)

Lists:

  • Ordered collections of items in square brackets [].
  • Store various data types within a list.
  • Access elements by index (position) starting from 0.
                # Create a list of numbers
                    numbers = [1, 2, 3, 4, 5]
                    
                    # Access the third element (index 2)
                    third_number = numbers[2]  # Value is 3
                    
                    # Add an element to the end of the list
                    numbers.append(6)
Functions
  • Reusable code blocks: Defined with def to perform specific tasks.
  • Arguments: You can provide inputs to functions.
  • Return Values: Functions can return outputs for further use.
                def greet(name):
                    """This function prints a greeting message."""
                    print("Hello,", name)
                
                # Call the function with an argument
                greet("Bob")
